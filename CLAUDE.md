Projeto: Tesselo - Procedural Geometry Puzzle1. Visão GeralTesselo é um jogo de puzzle minimalista baseado em tesselação e geometria espacial. O jogador deve preencher um grid dinâmico desenhando retângulos e quadrados que correspondam exatamente ao número contido em uma de suas células. O jogo é infinito, com geração procedural de níveis e dificuldade progressiva.2. Mecânicas Core & RegrasRegra de Área: Cada retângulo deve ter Largura x Altura = Número da Célula.Conexão Única: Cada forma deve conter exatamente UM número original do grid.Preenchimento Total: O nível só é vencido quando 100% das células do grid estiverem ocupadas sem sobreposição.Haptics: Vibração curta ao completar uma forma correta; vibração longa ao vencer o nível.4. Sistema de Dificuldade Progressiva (The Scaler)O tamanho do grid e a complexidade aumentam conforme o jogador avança:Níveis 1-5: Grid 4x4 (Tutoriais rápidos).Níveis 6-15: Grid 5x5 (Dificuldade padrão).Níveis 16-30: Grid 6x6 (Introdução de formas complexas).Níveis 31+: Grid 8x8 (Desafio Master).5. Algoritmo de Geração (Procedural Engine)O Claude Code deve implementar o TesseloGenerator:Partitioning: Divide a área total (ex: 25 células) em sub-retângulos aleatórios usando um método de Recursive Sub-divison.Shuffling: Para cada retângulo gerado, sorteia uma posição interna para colocar o número indicador.Validation: Garante que não existam áreas impossíveis de preencher ou formas que se cruzem.6.
